NBA2K24 is a basketball simulation game.
Your goal is to win a game as the New York Liberty versus the Minnesota Lynx.

Controls:
Pause = Start >    
Move = Left Stick
Sprint = Hold RT while moving

Offense:
Jump Shot = Hold X till meter fills
Pass = A  Rebound = Y

Defense:
Switch Player = A  Steal = X
Block/Rebound = Y
